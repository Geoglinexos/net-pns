# Net::PNS

protocol handler of PNS commands for PATLITE NH-FV series

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'net-pns'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install net-pns
